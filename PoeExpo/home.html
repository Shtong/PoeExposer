Welcome

The Path Of Exile Character Share Tool is a web-based tool that can be used to share your Path Of Exile character with other people with a simple link.

Your character data is safely isolated on the Path Of Exile servers. In order to share it you'll have to extract it from there and put it here. To do this, you can either install a plugin on your browser that will do all the work for you, or follow a 3-step guided manual process. Choose your preference below :

Install the browser plugin Manually import your character

This website has been developped by some PoE player; it has been built using Knockout and RequireJS.
Its source code is freely available on GitHub.
